# Mini-calculator
A calculator written in C for common mathematical operations. <br/>
Available operations are....<br/>

- Addition
- Subtraction
- Multiplication
- Division
- Remainder
- Square Root
- Power
- Sin()
- Cos()
- Tan()
- Cube Root
- Logarithm
- Inverse Sin()
- Inverse Cos()
- Inverse Tan()
- Factorial
